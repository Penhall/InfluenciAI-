# Setup de Ambiente — Desenvolvimento

Pré-requisitos:

- Windows 11, .NET 10 SDK (RC com include-prerelease), Visual Studio 2022 (ou Rider)
- PostgreSQL 15+, Redis, e Service Bus local (RabbitMQ) ou Azure Service Bus
- Docker Desktop (para subir dependências locais)

Serviços locais via Docker (exemplo):

```powershell
docker run -d --name pg -e POSTGRES_PASSWORD=postgres -p 5432:5432 postgres:15
docker run -d --name redis -p 6379:6379 redis:7
docker run -d --name rabbit -p 5672:5672 -p 15672:15672 rabbitmq:3-management
```

Configuração de secrets:

- Armazenar conexões/tokens via User Secrets (dev) e Key Vault (cloud)
- Padronizar nomes de variáveis e conexão por domínio
- `INFLUENCIAI_DB` (opcional) pode ser usado pelo design-time factory do EF durante migrações

Build e execução:

- Backend (API .NET 10): `dotnet build` → `dotnet run` nos serviços
- Desktop: WPF em .NET 8; seguir estilos `.rules/style_guide.md`

Observabilidade local:

- Habilitar logs estruturados e tracing (OpenTelemetry)
- HealthChecks UI disponível em `/health/ui`
