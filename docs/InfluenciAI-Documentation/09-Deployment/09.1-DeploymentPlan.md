# Plano de Deploy — Backend e Desktop

Backend (serviços):

- Ambientes: dev, staging, prod (isolados)
- Contêineres Docker; deploy em AKS (Azure Kubernetes Service)
- CI/CD: build, testes, scan, deploy progressivo; feature flags
- Configuração por ambiente via Key Vault; health checks e readiness/liveness
 - .NET 10 (atual) e atualização para GA quando disponível

Cliente Desktop (WPF):

- Empacotamento MSIX; atualização automática
- Configuração de endpoints por ambiente; telemetria habilitada

Go-live checklist (resumo):

- [ ] Observabilidade ativa (logs/metrics/traces)
- [ ] HealthChecks UI publicado e autenticado
- [ ] Secrets e RBAC conferidos
- [ ] Rollback e feature flags preparados
- [ ] Documentação operacional atualizada
